# Notification Rules

## Event Keys

Every configurable notification must use a real event key.

Never use an empty event key.

---

## Roles

Core roles:

- admin
- operator
- technician
- customer

Communication Matrix should represent role coverage clearly.

---

## Channels

SMS and In-App are channels.

Channels can be enabled/disabled per company and per role.

Company does not edit platform-owned template text.
