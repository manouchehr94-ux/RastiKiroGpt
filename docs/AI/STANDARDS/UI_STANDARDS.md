# UI Standards

## General

Use the existing design system.

Do not create one-off styles unless necessary.

Prefer shared CSS components.

---

## Numeric Display

Amounts must use thousands separators.

Prices, fees, wages, balances, totals, and discounts must be formatted consistently.

---

## Numeric Input

Phone, national ID, postal code and OTP must preserve leading zeros.

Use text input with numeric input mode where appropriate.

---

## Mobile

Technician pages must be mobile-first.

Input focus must not break page fit.

Avoid font-size below 16px on mobile inputs.

---

## Public Pages

Public pages must never render admin sidebar or dashboard layout.
