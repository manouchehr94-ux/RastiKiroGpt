# توضیح نسخه تمیز پروژه

این نسخه از آخرین کدهای New ساخته شده و patchهای سفارش تا فاز 19B داخل آن اعمال شده است.

## تغییرات مهمی که داخل این نسخه هست

- اصلاح visibility سفارش‌ها برای تکنسین‌ها بر اساس اولویت ۱/۲/۳
- تنظیم خودکار `priority2_visible_at` و `priority3_visible_at` هنگام ثبت سفارش
- جلوگیری از دیدن سفارش مخفی توسط تکنسین با حدس زدن آیدی
- آزاد شدن نام مشتری، شماره تلفن و آدرس در ثبت سفارش توسط ادمین
- حفظ ساختار SaaS و tenant isolation

## چیزهایی که عمداً حذف شده‌اند

- فایل‌های تست فازی مثل `tests_phase18d.py` و `tests_phase19a.py`
- دیتابیس محلی `db.sqlite3` برای جلوگیری از overwrite شدن دیتابیس سیستم شما
- `.env` برای جلوگیری از جابه‌جایی تنظیمات خصوصی
- پوشه `.git`
- فایل‌های cache پایتون

## روش سریع کار

برای هر تغییر کوچک لازم نیست ۴۰۰ تست اجرا شود. معمولاً این کافی است:

```powershell
python manage.py check
```

برای تغییرات سفارش:

```powershell
python manage.py test apps.orders
```

تست کامل را فقط آخر هر فاز اجرا کنید:

```powershell
python manage.py test apps.orders apps.tenants apps.accounts
```
