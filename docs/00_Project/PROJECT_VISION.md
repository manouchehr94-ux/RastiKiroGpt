# Project Vision

Rasti Service is a multi-tenant SaaS platform for Iranian service-dispatch companies.

The system helps each company manage:

- customers
- orders
- technicians
- invoices
- payments
- SMS/notifications
- reports
- future accounting modules

Rasti Service sells software and infrastructure to companies. It does not sell the field service to the customer.

## Long-term vision

Rasti Service should become a reliable operating system for service companies:
- simple for small companies
- scalable for many tenants
- safe for money-related operations
- extensible for accounting, inventory, CRM, mobile apps, and APIs
