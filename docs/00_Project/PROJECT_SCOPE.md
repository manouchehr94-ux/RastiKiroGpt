# Project Scope

## Current primary scope

1. Stable order lifecycle
2. Technician assignment and visibility
3. Invoice issuance and settlement snapshot
4. Payment safety
5. Multi-tenant isolation
6. SMS/notification stability
7. Internal admin/operator workflows

## Deferred scope

- Full accounting
- Inventory
- CRM
- Mobile app
- Public API marketplace
- Automated refunds
- Double-entry accounting
- Real PSP split settlement

Deferred does not mean forgotten. It means the architecture must be compatible, but implementation waits.
